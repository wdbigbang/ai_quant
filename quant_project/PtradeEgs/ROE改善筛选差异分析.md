# ROE改善筛选差异分析

> ⚠️ 重要技术文档，后续研究需要参考

---

## 发现时间

2026-04-07

---

## 问题描述

聚宽和PTrade在ROE改善筛选的排序结果不一致：

| 平台 | 前5只股票 |
|------|----------|
| 聚宽 | 002306, 002336, 002905, 002456, 002269 |
| PTrade | 002905, 002707, 002456, 002247, 002269 |
| 交集 | 002905, 002456, 002269（仅3只）|

---

## 根本原因

PTrade v3.1 新增了 **ROE异常过滤**: `|ROE| > 100%` 视为财务异常并过滤

---

## 典型案例：002306.XSHE

### 聚宽数据

| 季度 | ROE值 | 说明 |
|------|-------|------|
| 2022-09-30 | -3.71% | ROE_1 |
| 2022-12-31 | -24.35% | ROE_2 |
| 2023-03-31 | -52.55% | ROE_3 |
| 2023-06-30 | -344.97% | ROE_4 ⚠️ 超过100% |
| 2023-09-30 | 187.74% | ROE_5 ⚠️ 超过100% |

**改善值计算**：
```
increase = 4 * ROE_5 - ROE_1 - ROE_2 - ROE_3 - ROE_4
         = 4 * 187.74 - (-3.71) - (-24.35) - (-52.55) - (-344.97)
         = 1176.54
```

**聚宽排名**: 第1名

### PTrade处理

因 ROE_4 和 ROE_5 超过100%，被过滤，不参与改善值排序。

---

## 两种方案对比

### 方案A（当前采用）：保持异常过滤

- ✅ 优点：过滤财务异常股票，避免选入风险股
- ❌ 缺点：选股结果与聚宽不一致

### 方案B（备选）：去掉异常过滤，完全对齐聚宽

- ✅ 优点：选股结果一致
- ❌ 缺点：可能选入净资产接近零的财务异常股票

## 技术细节

### 单季度ROE计算公式

```python
# Q1单季度 = 一季报累计（直接用）
# Q2单季度 = 半年报累计 - 一季报累计
# Q3单季度 = 三季报累计 - 半年报累计
# Q4单季度 = 年报累计 - 三季报累计

# 单季度ROE = 单季度净利润 / 当期净资产
```

### ROE改善值计算公式

```python
increase = 4 * ROE_5 - ROE_1 - ROE_2 - ROE_3 - ROE_4

# 含义：最新季度权重最高(4倍)，减去前4个季度的ROE
# 改善值大 = 最新季度ROE显著高于前4季度平均水平
```
